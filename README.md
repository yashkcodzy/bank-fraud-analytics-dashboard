# 🏦 Bank Fraud Analytics Dashboard

## 📌 Project Overview

The Bank Fraud Analytics Dashboard is an interactive Power BI project designed to monitor banking transactions, detect fraud patterns, and provide business insights. The dashboard enables analysts to identify suspicious activities, analyze customer behavior, monitor merchant performance, and support fraud investigation through interactive reports.

---

# 🎯 Objective

- Monitor bank transactions.
- Detect fraudulent transactions.
- Analyze customer and merchant behavior.
- Identify fraud trends over time.
- Improve decision-making using interactive dashboards.
