# Bank Fraud Analytics Dashboard

## Project Overview
This project aims to analyze bank transactions and detect fraudulent activities using Power BI.

## Objectives
- Analyze banking transactions
- Detect fraudulent transactions
- Create an interactive dashboard
- Generate business insights using DAX

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Git & GitHub

## Dataset
Bank Transaction Fraud Detection Dataset (CSV)
